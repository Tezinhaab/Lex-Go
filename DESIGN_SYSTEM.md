# Lex GO - Sistema de Design

## Paleta de Cores Vibrantes

### Cores Principais
- **Electric Blue** (#007BFF) - Cor primária para ações e destaques
- **Solar Yellow** (#FFD93D) - Cor secundária para conquistas e alertas positivos
- **Mint Green** (#00E6A1) - Cor de acento para sucesso e progresso
- **Neon Pink** (#FF3CAC) - Cor para alertas e elementos de destaque
- **Ice White** (#F9F9F9) - Fundo principal
- **Bluish Gray** (#2C3E50) - Texto principal

## Tipografia

### Fontes
- **Primária**: Poppins (arredondada e moderna)
- **Alternativas**: Inter, Nunito Sans
- **Monoespaçada**: Geist Mono

### Hierarquia
- Títulos: Poppins Bold/Semibold
- Corpo: Poppins Regular
- Legendas: Poppins Light

## Estilo Visual

### Características
- **Flat Design** com ilustrações 2D vetoriais
- **Minimalista** com muito espaço em branco
- **Bordas arredondadas** (1rem / 16px)
- **Sombras suaves** para profundidade
- **Ícones simples** estilo vetor 2D

### Animações
- **Transições suaves**: fade, slide
- **Microinterações**: feedback de toque em botões
- **Duração**: 200-300ms para interações rápidas
- **Easing**: ease-in-out para naturalidade

## Mascote - Juriton

### Descrição
Tucano advogado com:
- Peruca de juiz europeu (branca)
- Óculos redondos
- Terno escuro com gravata
- Bico laranja característico
- Expressão amigável e profissional

### Uso
- Tela de login (canto inferior direito)
- Chat AI (avatar)
- Conquistas e celebrações
- Mensagens de boas-vindas

## Inspirações
- **Duolingo**: Gamificação e progressão visual
- **Notion**: Layout limpo e organizado
- **Figma Mobile**: Navegação intuitiva e moderna
